# JavaScript Arrays : what I learned 

## Problem 
I found so many problems when I learned JS at first. The syntax is different from Python. I only have some basic Python knowledge, but I learned step by step: functions, arrays, data types, commits, etc. 

## Cause 
For example , I can't understand what is array , so I learned and compare with python through some example as below . 

## Compare
```js
let fruits = ["Apple", "Orange", "Plum"];

console.log( fruits[0] ); // Apple
console.log( fruits[1] ); // Orange
console.log( fruits[2] ); // Plum
```

I found it works the same way as Python lists. One key difference is that JS arrays can store any type of object, but Python lists are more strict. 

```js
let arr = [ 'Apple', { name: 'John' }, true, function() { alert('hello'); } ];
alert( arr[1].name ); // John
arr[3](); // hello
```


## Takeaway
I can see the differences between the two languages and learn more interesting points when I compare them.
